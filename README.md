# Sales-Funnel-Operations-Performance-Dashboard
ElectricPe Lead-to-Delivery Funnel Analysis & Dashboard

Project Overview
This project was completed as part of the ElectricPe Business Analyst Internship assignment. The objective was to analyze the end-to-end lead-to-delivery funnel and build an Excel dashboard to identify operational insights and improvement opportunities.

Objectives

Analyze conversion across funnel stages

Measure turnaround time (TAT) at each stage

Identify cancellation patterns and operational gaps

Evaluate store performance and customer satisfaction

Analysis Performed

Conversion rates from Lead → Pre-booking → Booking → Delivery

Average turnaround time by stage and store

Cancellation trends by store and reason

Hot, Warm, and Cold lead distribution

Store-wise customer satisfaction (NPS)

Dashboard Features

Lead conversion funnel by store

TAT performance across stages

Cancellation trends and reasons

Lead quality distribution

Customer ratings visualization

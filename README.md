# Sales-Funnel-Operations-Performance-Dashboard
ElectricPe Lead-to-Delivery Funnel Analysis & Dashboard

Project Overview:<br>
This project was completed as part of the ElectricPe Business Analyst Internship assignment. The objective was to analyze the end-to-end lead-to-delivery funnel and build an Excel dashboard to identify operational insights and improvement opportunities.

Objectives<br>
• Analyze conversion across funnel stages<br>
• Measure turnaround time (TAT) at each stage<br>
• Identify cancellation patterns and operational gaps<br>
• Evaluate store performance and customer satisfaction

Analysis Performed<br>
• Conversion rates: Lead → Pre-booking → Booking → Delivery<br>
• Average turnaround time by stage and store<br>
• Cancellation trends by store and reason<br>
• Hot, Warm, and Cold lead distribution<br>
• Store-wise customer satisfaction (NPS)

Dashboard Features<br>
• Lead conversion funnel by store<br>
• TAT performance across stages<br>
• Cancellation trends and reasons<br>
• Lead quality distribution visualization<br>
• Customer ratings insights<br>
